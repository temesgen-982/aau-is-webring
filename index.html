Information Systems Class of 2026 Webring

Welcome to our digital yearbook! This webring connects the personal websites, portfolios, and blogs of the 2026 Information Systems graduating batch. Let's stay connected as we enter the tech industry.

How to Join

1. Copy the widget code below and paste it into your website's footer:

<div id="aau-is-webring"></div>
<script src="https://YOUR_GITHUB_USERNAME.github.io/aau-is-2026-webring/embed.js"></script>

(Note: Remember to change YOUR_GITHUB_USERNAME to your actual GitHub username in the script source!)

2. Submit a Pull Request modifying webring.json to add your name and website URL, or text the admin directly.
